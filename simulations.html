01 · What it does

Three capabilities, composable.

We built around three things regulated organizations consistently need: exact reproducibility, "what if" branching, and a traceable path from cause to outcome.

01

Exact replay

Start a simulation with the same initial state and it replays identically, every time, on any supported hardware. Each run produces a signed record of inputs, agent definitions, and interactions.

02

"What if" branching

Pause at any step, change a single variable, and continue both versions. Compare them at the end or anywhere in between to see exactly what your change caused.

03

Mix agent types

Combine different kinds of agents in one simulation: large-language-model reasoners, rule-based bots, reinforcement-learning agents, or handcrafted actors. Switch the underlying model (OpenAI, Anthropic, or locally-hosted) per agent.

03 · SDK

An SDK that gets out of the way.

# define agents, environment, policy. run at scale
from bitqubic import Simulation, Agent, replay

sim = Simulation(name="macroprudential-v12", seed=0xB17Q)
sim.populate(Agent.from_manifest("cohorts/banks.yaml"), n=12_000)
sim.attach_environment("basel-iv-2026q1")

with sim.run(steps=90, parallel_branches=400) as run:
    run.fork_at(t=31, mutate={"rate_hike_bps": 75})

report = replay(run.manifest).attribute("insolvencies")
# -> signed, reproducible, regulator-ready
